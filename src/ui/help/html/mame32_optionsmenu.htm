Options Menu

       Game List Font :  Selecting this option from the options menu will change the font used for the GUI display.

       Game List Clone Color :  Selecting this option from the options menu will change the font color used for the GUI display of clones in the 'Grouped' game list view.

       Directories : Selecting this option from the options menu will allow for choosing a different directory for a number of Mame32 support files including: ROMs, samples, config, high scores, snapshots, input files, state, artwork, memory card, flyers, cabinets, marquees, and nvram.  Normally these directories are created under the Mame32 executable’s directory.

       Default Options : Selecting this option from the options menu will allow for the global customization of all games in the game list.  These settings are overridden by altering the individual game settings using the File • Properties menu item.

       Reset to Default : Selecting this option from the options menu will allow for resetting various parts of Mame32 to their defaults.  Mame32 will prompt before resetting, giving the option to cancel.

       Interface Options : Selecting this option from the options menu will allow:
1. Toggling on or off the search for new games on start.
2. Toggling on or off the ability to use a game pad to navigate the GUI.
3. Toggling on or off Mame32 broadcasting the selected game programmatically to other listening applications like message clients, or an LED message bar,
4. Selecting a random image from the \bkground directory on startup.
5. Turning off the two information screens once a game is launched.
6. The cursor to be hidden.
7. Increasing the Windows thread priority for Mame32 giving it more CPU and resources slices.
8. Allowing the art pane images to cycle at a certain interval [in seconds].
9. Stretch screenshot to larger size, this will enlarge the items in the art pane scaling them upwards depending on the size of the GUI.
10. Selecting inheritable filters, this allows parent folders to set the filters for their children sub-folders.
11. Globally clear the play time and number of times played counters.
12. Toggling on or off the ability to use a keyboard encoder solution to navigate the GUI and launch games [helpful in Mame cabs].
13. No offset for clones missing parent in view toggle will cause clones not be indented if the parent game is not present due to current folder view, missing ROMs, or filtering.

       Background Image : Selecting this option from the options menu allows the choosing of a background image from the \bkground directory.  This image will be displayed beneath the three window panes in the GUI.

       Language : Selecting this option from the options menu allows the choosing of an externally supported *.lng file.  These files translate the internal Mame32 menu system [accessed from the tab key while playing a game] into a different language.
