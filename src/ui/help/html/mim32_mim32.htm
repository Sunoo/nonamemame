What is MadeInMame32

MadeInMame32 is a derivative of Mame32. It uses all of his resources (code, artwork) but aims to add a few useful features that won’t be included in Mame32.

 

It implies :

       A new MadeInMame32 release does not necessarily mean new features ;

       Mame32 licence applies with no restriction, in particular the “as is” distribution ;

       Since MadeInMame32 is a derivative of Mame32, it will never include any feature required by Mame32 Dev Team, unless being authorized.

 

There is no enhancement regarding Mame core.
