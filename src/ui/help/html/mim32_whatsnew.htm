Changes to MadeInMame32

 

These changes are in addition to Mame32 releases.

v.81

  • Added a password to prevent kids from playing user-defined games (erotic, violent, …).
  • Added a splash screen displayed while waiting for loading on slow machines.
  • Added a popup menu to the system tray icon to manage a few MadeInMame32 shortcuts (game selection, hide/show windows, exit).

v.80

  • Added support to M.A.S.H’s PCB info in an extra window.
  • Added the possibility to load the background from predefined internal images.

v.79

  • Added an option in tree context menu to choose whether tree root directories are sorted by alphabetic order or internal creating order.
  • Added new folders : BIOS, screens sizes, number of colours, vertical refresh rate, light gun, stereo and samples.
  • Added some new icons : tree folders, ROMs, options.

v.67

  • Added an default internal background.
  • Added a feature to save the current displayed game list (detail or grouped view mode) to a formatted or tabulated text file.
  • Added an icon in the task bar to start the current selected game, and hide/show Mame32 window.
